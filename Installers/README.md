Navigate to folder /opt/ (usually 2levels above \~/)

mkdir ./installers && cd installers 

Direction: 

Each installer file is intended to have the script in it copied and pasted directly into the terminal directory at the location wished for them to be located. (opt/installers).

the script will create the .sh file in the directory you are currently pasting it into and it will also take care of making the shell script executable.

Using the scripts:

After pasting the block into the terminal- run  ./name of file.sh and press enter. The application will automatically begin the install process for its setting up in the environment. 


Follow the install process instructions as it runs and provide added details if prompted.
